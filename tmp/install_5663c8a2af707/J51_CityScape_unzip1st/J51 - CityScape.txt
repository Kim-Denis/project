Enjoy!


NOTE: To Setup Horizontal MooMenu
---------------------------------
Go to the admin panel >> Module Manager >> Click on your menu module (Main Menu, Top Menu.....)
Under "Module Parameters" set "Menu Style" to "List"
Under "Module Parameters" change "Always show sub-menu Items" to "yes"
Under "Details" change the "position" to "hornav"
Under "Advanced Parameters" set "Menu Class Suffix" to "_horiz"
